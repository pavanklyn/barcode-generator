# Barcode Generator

A simple Python project to generate cyberpunk-style barcodes using `python-barcode` and `Pillow`.

## Features

* Generate Code128 barcode
* Save barcode as PNG image
* Invert barcode colors using Pillow
* Beginner-friendly Python project

## Technologies Used

* Python
* python-barcode
* Pillow (PIL)

## Installation

Install required packages:

```bash
pip install python-barcode pillow
```

## Run the Project

```bash
python "Cyberpunk Barcode.py"
```

## Output

The project generates:

* `cyber_barcode.png`
